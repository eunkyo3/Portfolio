## Eunkyo's Portfolio

시큐어 코딩, 클라우드 인프라, AI 프로젝트 경험을 정리한 개인 백엔드 포트폴리오 사이트입니다.  
보안 관점에서 시작해 서버·데이터·인프라 전반을 아우르는 경험을 한 눈에 볼 수 있도록 구성했습니다.

---

### 주요 섹션

- **Home**  
  - 이름/포지션/한줄 소개가 포함된 히어로 섹션  
  - GitHub · Notion 아카이빙 카드  
  - 상단 네비게이션에서 각 섹션으로 부드럽게 스크롤 이동

- **About**  
  - 이름, 포지션(백엔드 개발자), 학력, 연락처, 이메일, 관심 분야  
  - 성장 배경과 개발 가치관을 담은 자기소개

- **Experience**  
  - `애니셀`에서의 보안 솔루션·게이트웨이·관제 대시보드 관련 백엔드 경험  
  - 다단계 통신 아키텍처, MQTT/WebSocket/RESTful API 기반 실시간 데이터 처리

- **Projects**  
  - **수어 인식 서비스 실험 (SignGPT 기반)** – 수어 접근성 서비스 아이디어  
  - **AI 뉴스 기반 종목 추천 시스템** – KoBERT 감성 분석, DART 연동 ([코드](https://github.com/eunkyo3/stock-news-ai))  
  - **연락처 관리 앱 (Node.js & Sequelize)** – RESTful CRUD, Sequelize ORM ([코드](https://github.com/eunkyo3/contact-app))  
  - **메모 관리 웹 애플리케이션 (Java)** – Gradle 기반 Java 웹 앱 ([코드](https://github.com/eunkyo3/memo))  
  - **Flutter 앱 모음 포트폴리오** – 다양한 플러터 앱 데모 모음 ([사이트](https://eunkyo3.github.io/))

- **Skills**

  - 정보처리기사/산업기사, SQLD, 웹디자인개발기능사, SW개발_L3_22V2  
  - 사이버가디언즈, 화이트해커 양성교육, AI 슈퍼컴퓨터 캠프, AI Youth Challenge(교육부 장관상), 디지털새싹 메이커톤(최우수상) 등

### 기술 스택

- **Frontend**: React, TypeScript, Vite, Tailwind CSS  
- **State / Routing**: React Router, React Context (Theme, I18n)  
- **Backend 경험**: Java(Spring), Node.js(Express), Sequelize, MySQL, Oracle, Redis  
- **Infra / DevOps**: AWS, Docker, Kubernetes, Linux, Vercel

---

### 로컬 실행 방법

```bash
git clone https://github.com/eunkyo3/portfolio.git
cd portfolio
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 에 접속하면 포트폴리오를 확인할 수 있습니다.
