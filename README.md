## Eunkyo's Portfolio

### 기술 스택

- **Frontend**: React, TypeScript, Vite, Tailwind CSS  
- **State / Routing**: React Router, React Context (Theme, I18n)  
- **Backend 경험**: Java(Spring Boot), Node.js(Express), Sequelize, Python 기반 AI 서비스 연동, MySQL, PostgreSQL, Oracle, Redis, Firebase  
- **Infra / DevOps**: AWS(ECS, EB, 인프라 구축), Docker, Kubernetes, Linux, Vercel  
- **AI / Data**: KoBERT, OpenVINO, Computer Vision, LLM 연동, 데이터 파이프라인 설계 경험

### 로컬 실행 방법

```bash
git clone https://github.com/eunkyo3/Portfolio.git
cd Portfolio
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 에 접속하면 포트폴리오를 확인할 수 있습니다.
